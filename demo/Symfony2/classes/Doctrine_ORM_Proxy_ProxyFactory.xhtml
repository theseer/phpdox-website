Doctrine\ORM\Proxy\ProxyFactory

This factory is used to create proxy objects for entities at runtime.

  • Author: Roman Borschel <roman@code-factory.org>
  • Author: Giorgio Sironi <piccoloprincipeazzurro@gmail.com>
