\PhpIniRequirement

Represents a PHP requirement in form of a php.ini configuration.

  • Author: Tobias Schultze <http://tobion.de>

Constructor

  • public__construct( string $cfgName, Boolean|callback $evaluation, Boolean $approveCfgAbsence, string|null $testMessage, string|null $helpHtml, string|null $helpText, Boolean $optional )

  • Constructor that initializes the requirement.

      • Parameters:

      • $cfgName - The configuration name used for ini_get()
      • $evaluation - Either a Boolean indicating whether the configuration should evaluate to true or false,
      • $approveCfgAbsence - If true the Requirement will be fulfilled even if the configuration option does not exist, i.e. ini_get() returns false.
      • $testMessage - The message for testing the requirement (when null and $evaluation is a Boolean a default message is derived)
      • $helpHtml - The help text formatted in HTML for resolving the problem (when null and $evaluation is a Boolean a default help is derived)
      • $helpText - The help text (when null, it will be inferred from $helpHtml, i.e. stripped from HTML tags)
      • $optional - Whether this is only an optional recommendation not a mandatory requirement
