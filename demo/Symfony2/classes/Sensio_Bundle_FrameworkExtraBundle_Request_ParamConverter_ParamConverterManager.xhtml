  • publicadd( Sensio\Bundle\FrameworkExtraBundle\Request\ParamConverter\ParamConverterInterface $converter, integer $priority, string $name )

  • Adds a parameter converter.
    Converters match either explicitly via $name or by iteration over all
converters with a $priority. If you pass a $priority = null then the
added converter will not be part of the iteration chain and can only
be invoked explicitly.

      • Parameters:

      • $converter - A ParamConverterInterface instance
      • $priority - The priority (between -10 and 10).
      • $name - Name of the converter.
