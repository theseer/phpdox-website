  • Returns a cache key for the current asset.
    The key is composed of everything but an asset's content:

 * source root
 * source path
 * target url
 * last modified
 * filters

      • Parameters:

      • $asset - The asset
      • $additionalFilter - Any additional filter being applied
      • $salt - Salt for the key

      Returns:

      • A key for identifying the current asset
