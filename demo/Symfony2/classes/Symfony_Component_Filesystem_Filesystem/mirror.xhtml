Symfony\Component\Filesystem\Filesystem::mirror

Mirrors a directory to another.

    Signature

    public function mirror(string $originDir, [Traversable $targetDir = null, [Traversable $iterator = null, array $options]] )

    Parameters

    $originDir — string
    The origin directory
    $targetDir — string
    The target directory
    $iterator — \Traversable
    A Traversable instance


    $options — array
    An array of boolean options
    Valid options are:
    - $options['override'] Whether to override an existing file on copy or not (see copy())
    - $options['copy_on_windows'] Whether to copy files instead of links on Windows (see symlink())
    - $options['delete'] Whether to delete files that are not in the source directory (defaults to false)

    Errors/Exceptions

    IOException
    When file type is unknown
