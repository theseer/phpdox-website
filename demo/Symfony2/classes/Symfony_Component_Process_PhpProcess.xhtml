Symfony\Component\Process\PhpProcess

PhpProcess runs a PHP script in an independent process.

$p = new PhpProcess('<?php echo "foo"; ?>');
$p->run();
print $p->getOutput()."\n";

  • Author: Fabien Potencier <fabien@symfony.com>
