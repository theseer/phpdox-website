  • truepublicenable( integer $errorReportingLevel, Boolean $displayErrors )

  • Enables the debug tools.
    This method registers an error handler and an exception handler.

If the Symfony ClassLoader component is available, a special
class loader is also registered.

      • Parameters:

      • $errorReportingLevel - The level of error reporting you want
      • $displayErrors - Whether to display errors (for development) or just log them (for production)
