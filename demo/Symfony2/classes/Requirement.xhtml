\Requirement

Represents a single PHP requirement, e.g. an installed extension.

It can be a mandatory requirement or an optional recommendation.
There is a special subclass, named PhpIniRequirement, to check a php.ini configuration.

  • Author: Tobias Schultze <http://tobion.de>

      • $fulfilled - Whether the requirement is fulfilled
      • $testMessage - The message for testing the requirement
      • $helpHtml - The help text formatted in HTML for resolving the problem
      • $helpText - The help text (when null, it will be inferred from $helpHtml, i.e. stripped from HTML tags)
      • $optional - Whether this is only an optional recommendation not a mandatory requirement
