Symfony\Component\PropertyAccess\PropertyPathInterface

A sequence of property names or array indices.

  • Author: Bernhard Schussek <bschussek@gmail.com>

Methods

  • public__toString( )

  • Returns the string representation of the property path

      • Returns:

      • The path as string

  • publicgetElement( integer $index )

  • Returns the element at the given index in the property path

      • Parameters:

      • $index - The index key

      Returns:

      • A property or index name

      Throws:

      • Exception\OutOfBoundsException If the offset is invalid

  • publicgetElements( )

  • Returns the elements of the property path as array

      • Returns:

      • An array of property/index names

  • publicgetLength( )

  • Returns the length of the property path, i.e. the number of elements.

      • Returns:

      • The path length

  • publicgetParent( )

  • Returns the parent property path.
    The parent property path is the one that contains the same items as
this one except for the last one.

If this property path only contains one item, null is returned.

      • Returns:

      • The parent path or null

  • publicisIndex( integer $index )

  • Returns whether the element at the given index is an array index

      • Parameters:

      • $index - The index in the property path

      Returns:

      • Whether the element at this index is an array index

      Throws:

      • Exception\OutOfBoundsException If the offset is invalid

  • publicisProperty( integer $index )

  • Returns whether the element at the given index is a property

      • Parameters:

      • $index - The index in the property path

      Returns:

      • Whether the element at this index is a property

      Throws:

      • Exception\OutOfBoundsException If the offset is invalid
